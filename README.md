
# hw-22.Django — Интернет-магазин Skystore

## Описание
Проект интернет-магазина **Skystore** реализован с использованием фреймворка Django. Цель проекта — создать базовую структуру онлайн-магазина с возможностью отображения домашней страницы и контактной информации.

### Основные функции:
- Отображение главной страницы с описанием сервиса.
- Страница контактов с формой обратной связи (https://example.com).
- Добавлена возможность добавлять товары
- Появилась возможность вести блог и оставлять отзывы
- Добавлена функция регистрации и авторизации, работа с почтой
- Добавлены права доступа и группы

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/hw-22.Django.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd hw-22.Django
   ```
3. Активируйте виртуальное окружение и установите зависимости:
   ```bash
   poetry install
   ```
4. Примените миграции:
   ```bash
   python manage.py migrate
   ```
5. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```

Откройте браузер и перейдите по адресу: [http://127.0.0.1:8000](http://127.0.0.1:8000).

## Используемые технологии
- Python 3.13+
- Django 5.2.1
- Bootstrap (CDN)
- Poetry (менеджер зависимостей)
- PostgreSQL 17
- Redis
